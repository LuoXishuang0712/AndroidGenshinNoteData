**项目简介**

这个项目是一个安卓APP，它可以将原神游戏内的数据（通过米游社或Hoyolab的api获取），并显示在APP内的列表与桌面的小部件上。
同时，它也提供了一个简单易用的树脂恢复时间计算器。

**功能**

* 支持多个账号
* 支持多个桌面小部件，并能同步更新
* 断网的情况下仍能计算树脂恢复时间
